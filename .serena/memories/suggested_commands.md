# Suggested Commands

## Development
- Install dependencies: `pip install -r requirements.txt`
- Install dev dependencies: `pip install -r requirements-dev.txt`
- Run application (Local): `python app.py` (Default port: 5000 or 5002 depending on config, see app.py)
- Run application (Docker): `docker-compose up -d`

## Testing
- Run tests: `pytest`
