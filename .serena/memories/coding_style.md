# Coding Style and Conventions
- **Language**: Python (Backend), JS/HTML/CSS (Frontend)
- **Comments/Docstrings**: Written in Japanese.
- **Style**:
  - Python: Standard PEP8-like. Snake_case for functions/variables. PascalCase for classes.
  - Structure: Application Factory Pattern (`create_app`).
  - Blueprints used for routing.
- **Frontend**:
  - Tailwind CSS used via CDN.
  - Jinja2 templating.
