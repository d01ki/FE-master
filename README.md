## 🚀 クイックスタート

```bash
# リポジトリをクローン
git clone https://github.com/d01ki/FE-master.git
cd FE-master

# 仮想環境を作成
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 依存パッケージをインストール
pip install -r requirements.txt

# アプリケーションを起動
python app.py
```

🎉 http://localhost:5002 でアクセスできます！
