# 🎓 基本情報技術者試験 学習アプリ (FE Master)

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Python](https://img.shields.io/badge/python-3.8+-blue.svg)
![Flask](https://img.shields.io/badge/flask-2.3+-green.svg)

基本情報技術者試験（FE）の学習を効率的にサポートする、モダンなWebアプリケーションです。Flask + SQLite + Tailwind CSSで構築されており、年度別過去問の管理から模擬試験まで包括的な学習機能を提供します。

## ✨ 主な機能

### 🎯 学習機能
- **ランダム練習**: 全問題からランダムに出題される練習モード
- **年度別模擬試験**: 2025年春期、2024年秋期など年度別の本番形式模擬試験
- **ジャンル別学習**: ハードウェア、ソフトウェア、ネットワークなど分野別練習
- **学習履歴管理**: 詳細な解答履歴と正答率の分析

### 📊 ダッシュボード
- **統計表示**: 学習進捗、正答率、ジャンル別成績の可視化
- **モダンUI**: Tailwind CSSによる美しいデザインとアニメーション
- **レスポンシブ**: スマートフォンからデスクトップまで対応

### 🔒 管理機能
- **パスワード保護**: 管理画面への安全なアクセス制御
- **年度別ファイル管理**: `2025r07_kamoku_a_spring.json` 形式での問題管理
- **JSON アップロード**: 年度・期別の問題データの簡単インポート
- **データベース管理**: 問題データと学習履歴の一括管理

## 🚀 クイックスタート

### 前提条件
- Python 3.8+
- pip (Python パッケージマネージャー)

### インストールと起動

```bash
# リポジトリをクローン
git clone https://github.com/d01ki/FE-master.git
cd FE-master

# 依存関係をインストール
pip install -r requirements.txt

# アプリケーションを起動
python run.py

# または、カスタム設定で起動
python run.py --host 0.0.0.0 --port 8080 --debug
```

アプリケーションは `http://localhost:5000` で起動します。

## 📚 使用方法

### 1. 問題データの準備

年度別の問題データをJSON形式で準備します。ファイル名は以下の命名規則に従ってください：

```
{年度}r{令和年度}_kamoku_a_{春期|秋期}.json
```

**例:**
- `2025r07_kamoku_a_spring.json` - 令和7年度（2025年）春期
- `2024r06_kamoku_a_autumn.json` - 令和6年度（2024年）秋期

### 2. JSON形式の例

```json
[
  {
    "question_id": "Q001",
    "question_text": "コンピュータのCPUが1秒間に実行できる命令数を表す指標は何か。",
    "choices": {
      "ア": "クロック周波数",
      "イ": "MIPS（Million Instructions Per Second）",
      "ウ": "キャッシュサイズ",
      "エ": "メモリ容量"
    },
    "correct_answer": "イ",
    "explanation": "MIPS（Million Instructions Per Second）は、CPUが1秒間に実行できる命令数を百万単位で表した性能指標です。",
    "genre": "ハードウェア"
  }
]
```

### 3. 問題データのアップロード

1. `http://localhost:5000/admin/login` にアクセス
2. パスワード `fe2025admin` でログイン
3. 管理画面でJSONファイルをアップロード
4. 必要に応じてデータベースに読み込み

### 4. 学習開始

- **ダッシュボード**: 学習の進捗確認
- **ランダム練習**: 基礎力向上
- **模擬試験**: 年度別の本番形式テスト
- **学習履歴**: 弱点分析と復習
